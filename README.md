# GreedySnake(贪吃蛇)
&nbsp;&nbsp;&nbsp;&nbsp;本程序是用 **C#** 语言和 **WPF** 技术开发。其界面简洁，操作简单（目前支持键盘操作蛇，*尚未实现蛇自动吃食物算法*）。玩家操控一条蛇在画布(**<font color=red>Canvas</font>**)上游走，游戏期间会生成不同的物品：  
1. 珠块。不同颜色的珠块有不同的积分。共有 **8** 种颜色(<font color=white>白</font>、<font color=red>红</font>、<font color=orange>橙</font>、<font color=yellow>黄</font>、<font color=green>绿</font>、<font color=blue>蓝</font>、<font color=purple>紫</font>)。  
2. 砖墙。蛇头碰到砖墙游戏会结束。  
3. 炸弹。吃掉炸弹会炸毁除了蛇之外所有物品。  
4. 钻石。价值分最高的物品，在游戏运行期间会不定时产生。  
5. 药水。这是会减积分的药水，可利用炸弹进行破坏掉（当然也会破坏其它物品）。
</br>
</br>
&nbsp;&nbsp;&nbsp;&nbsp;This program is developed using **C#** language and **WPF** technology. It's interface is simple and easy to operate. The player controls a snake to walk on the **<font color=red>Canvas</font>**, and during the activity, you will encounter different items:  
1. Bead. The food of the greedy snake is replaced by beads, and beads of different colors have different values. There are **8** colors.(<font color=white>white</font>、<font color=red>red</font>、<font color=orange>orange</font>、<font color=yellow>yellow</font>、<font color=green>green</font>、<font color=blue>blue</font>、<font color=purple>purple</font>)。  
2. Wall. The snake head hitting the brick wall will end the game.  
3. Bomb. Eating a bomb will blow up everything except snake.  
4. Diamond. The item with the highest value in the game will be generated irregularly during the game's runtime.  
5. Potion. This is a potion that will reduce game score and can be destroyed using a bomb (of course, it will also damage other items).
